關於我

我是宋祐嘉（Ken），一位全端取向的軟體工程師。對寫程式的興趣始於大學，之後在台大資訊培訓班與緯育課程持續精進，從 C++ 資料結構與演算法，一路延伸到 Java 後端、Python 應用與前端框架。

我最大的特質是自我學習能力——能快速吸收新技術，並把它實際做出成果。在校期間透過畢業專題與產學合作把技術落地，並以 IoT 智能門禁專題拿下 2022 高雄發明展金牌。
